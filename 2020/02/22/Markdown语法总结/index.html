Markdown语法总结

无目 发布于 2020-02-22

　　我经历了几天的学习，终于将Markdown的语法初步掌握了。那么接下来就是我对之前学习的一些总结（记性差没办法）:

一、标题

1.一级标题
============
2.二级标题
------------

或者

#一级标题
##二级标题
###三级标题
####四级标题
（可表示1-6级标题）

二、段落格式

1.段落

在段落末使用两个以上的空格加回车 或 在段落后面使用一个空行来表示重新开始一个段落。

PS:中国人有着首行缩进的习惯，这里我找到了两种方案，一种是将输入法切换到全角，然后再双击空格键（当然，我个人觉得有些麻烦）；另一种是直接用键盘打出来(如下所示）：

不断行的空白格&nbsp;或&#160;

2.字体

*斜体*
_斜体_
**粗体**
__粗体__
***粗斜体***
___粗斜体___

3.分隔线

***
* * *
*****
- - -
----------
(可以在星号与减号中间输入空格)

4.删除线

~~文本~~

老好用了(滑稽)

5.下划线

<u>文本</u>

6.脚注
暂时没弄明白。。。

三、列表

  1. 无序列表：
    • 无序列表使用星号(或加号减号)作为列表标记:
    • * 第一项
      * 第二项
      * 第三项
  2. 有序列表：
    • 1. 第一项
      2. 第二项
      3. 第三项
  3. 列表嵌套：
    • 1. 第一项：
      　　- 第一项嵌套的第一个元素
      　　- 第一项嵌套的第二个元素3.
      2. 第二项：
      　　- 第二项嵌套的第一个元素
      　　- 第二项嵌套的第二个元素
    • 需在子列表中的选项添加四个空格
  4. 之前的文本都是使用区块来划分的,这里我特地用列表来规划(笑)。

四、区块

1.引用

‘>’ + ‘空格’

2.嵌套

> 最外层
>> 第一层嵌套
>>> 第二层嵌套
依此类推

五、代码

1.函数片段

` import`函数

2.代码区块

使用四个空格或一个Tab键（记事本无法使用）

也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）

六、链接

1.链接使用方式

·[链接名称](链接地址)
Pixiv
或
·<链接地址>
https://www.pixiv.net/

1.高级链接

链接也可以用变量来代替，文档末尾附带变量地址：
      这个链接用 1 作为网址变量 [Google][1]
      这个链接用 runoob 作为网址变量 [Pixiv][2]
      然后在文档的结尾为变量赋值（网址）
   [1]: http://www.google.com/
   [2]: https://www.pixiv.net/
